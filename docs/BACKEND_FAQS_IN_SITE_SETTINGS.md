# FAQs في إعدادات الموقع

الأسئلة الشائعة التي تظهر في صفحة العرض (`/offers/:id`) أصبحت قابلة للتحرير من
لوحة الإدارة → إعدادات الموقع.

## الواجهة الأمامية
- تُحفظ كـ `faqs: Array<{ q: string; a: string }>` ضمن جسم إعدادات الموقع.
- تُرسل عبر `PUT /admin/site` مع باقي الإعدادات.
- تُقرأ من `GET /site` (publicApi.getSiteSettings) في صفحة العرض.

## المطلوب من الباك إند
1. قبول وحفظ المفتاح `faqs` كـ JSON array في جدول/مخزن إعدادات الموقع
   (نفس مكان `cancellationTerms`).
2. إعادته كما هو في الاستجابة العامة `GET /site` و `GET /admin/site`.
3. لا حاجة لإندبوينت جديد — فقط امتداد لإعدادات الموقع الحالية.

### مثال للجسم المرسل
```json
{
  "name": "خصومات",
  "faqs": [
    { "q": "هل أحتاج لتأكيد الحجز؟", "a": "لا، الباركود كافٍ." },
    { "q": "كيف يتم الدفع؟", "a": "مدى/فيزا/Apple Pay." }
  ]
}
```
